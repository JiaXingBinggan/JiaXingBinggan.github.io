极验与vue的原生js结合

发表于 2017-07-12 | 分类于 vue

既然说了极验，那么我就简单介绍一下极验是一个什么工具：

极验验证是一种在计算机领域用于区分自然人和机器人的，通过简单集成的方式，为开发者提供安全、便捷的云端验证服务。

与以往传统验证码不同的是，极验通过分析用户完成拼图过程中的行为特征，通过数据分析来判断是人还是机器。用户不必面对眼花缭乱的英文字符或汉字，整个验证过程变的像游戏一样有趣。目前极验在各大门户网站的登录页面上频繁出现，可见其在智能验证码方向的优秀。

现在我们来说说如何将极验前端的内容和vue用原生js进行结合：

一般来说，在jquery框架或者其它框架内，直接引入极验前端的sdk或js文件即可，并且极验的js文件是一个闭包函数，直接引入即可；但是在vue内想通过script标签引入第三方js文件就不像传统的前端框架或html文件那样方便，因为vue本身是使用的es6语法，需要通过用webpack和gulp之类的工具编译es6后再使用script，因此咱们还是中规中矩用点简单的方法来引入：

将极验的闭包函数赋值给一个变量，然后通过es6的export default导出（当然也可以直接将该闭包函数通过export default导出）

var gtInit = (function (global, factory) {
    "use strict";
    ...
})(typeof window !== "undefined" ? window : this, function (window, noGlobal) {
    "use strict";
    if (typeof window === 'undefined') {
        throw new Error('Geetest requires browser environment');
    }
    ...
    window.initGeetest = initGeetest;
    return initGeetest;
});

export default {
    gtInit
}

然后就可以在你想使用极验验证码的地方，通过import导入极验的闭包函数，由于闭包函数将想要返回的initGeetest函数赋传给了window，因此可以通过调用window.initGeetest({})函数来使用极验了：

 import gtInit from '../assets/gt'
 export default {
    ...
    initGtCaptcha () {
      var _this = this
      captchaApi.getGtCaptcha()
        .then(function(res) {
          console.log(gtInit);
          window.initGeetest({
              // 以下 4 个配置参数为必须，不能缺少
              gt: res.data.gt,
              challenge: res.data.challenge,
              offline: !res.data.success, // 表示用户后台检测极验服务器是否宕机
              new_captcha: res.data.new_captcha, // 用于宕机时表示是新验证码的宕机

              product: "float", // 产品形式，包括：float，popup
              width: "260px"
          }, function (captchaObj) {
            captchaObj.appendTo('#captcha');
            captchaObj.onReady(function () {
                document.getElementById('wait').style.display = 'none';
            });
            captchaObj.onSuccess(function () {
              _this.gtCapValid = captchaObj.getValidate();
            })
            captchaObj.onError(function () {
                // 出错啦，可以提醒用户稍后进行重试
              _this.$notify.error({
                title: '错误',
                message: '出错啦，请稍后进行重试'
              })
            });
          });
        })
    }
  }
}

window.initGeetest的回调函数的参数是极验验证码的dom，在该回调函数内可将验证码dom放置在你的页面任何地方，然后你就可以看到你的vue页面出现了极验验证码啦~（具体的函数、参数等内容请在注册极验验证码帐号后查看官方文档）。
